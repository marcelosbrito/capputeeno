### Frontend

1. Clone this repo

```bash
git clone https://github.com/marcelosbrito/reward-store.git
```

2. Go to the project folder

```bash
cd reward-store
cd reward
```

3. Install dependencies

```bash
npm install
```

3. Run Frontend

```bash
npm run dev
```

### Running the API

2. Go to the API folder

```bash
cd reward-store
cd api
```

3. Install dependencies

```bash
npm install
```

3. Run start

```bash
npm start
```
